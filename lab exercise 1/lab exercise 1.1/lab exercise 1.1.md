The purpose of this setup is to create a running light effect where the LEDs illuminate one after another from left to right, then reverse direction from right to left—similar to the classic “Knight Rider” or “Larson scanner” pattern. Each LED is connected to a digital pin through a resistor to control current flow safely.

The Arduino program uses a loop structure to turn LEDs on and off in sequence, often with a short delay between transitions to create smooth motion. This exercise helps learners understand looping logic, array indexing, and timing control in Arduino programming while reinforcing basic circuit design principles like grounding and current limitation.

It’s a simple yet visually engaging way to explore how software can control hardware behavior dynamically.
